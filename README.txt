# دعوة زفاف أنس ومريم — النسخة Premium

## الملفات
- `index.html` الصفحة الرئيسية
- `style.css` التصميم والألوان
- `script.js` الحركة، الموسيقى، العد التنازلي، والنموذج
- `assets/images/cover-placeholder.svg` صورة مؤقتة
- `assets/music/wedding.mp3` موسيقى أصلية مؤقتة بدون كلمات

## الرفع إلى GitHub
ارفع **كل الملفات والمجلدات كما هي** إلى نفس الـ Repository.
يجب أن يكون `index.html` في الصفحة الرئيسية للمستودع.

## تغيير الصورة
ضع صورتكم داخل:
`assets/images/`

ثم غيّر هذا السطر في `index.html`:
`assets/images/cover-placeholder.svg`

إلى اسم الصورة الجديدة، مثلاً:
`assets/images/anas-maryem.jpg`

## تغيير الموسيقى
ضع ملف MP3 الجديد داخل:
`assets/music/`

وسمّه:
`wedding.mp3`

بهذه الطريقة لا تحتاج لتعديل الكود.

## تغيير الألوان
افتح `style.css` وعدّل القيم الموجودة في أول الملف تحت:
`:root`

## ملاحظة RSVP
النموذج حالياً تجريبي ويحفظ الرد على جهاز الضيف فقط.
لجمع الردود فعلياً، يجب ربطه لاحقاً بـ Google Sheets أو خدمة Form.
